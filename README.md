# 📊 Pharmacy-Sales-Performance-Dashboard

## 📌 Project Overview
This project focuses on analyzing pharmacy data from **120+ branches across Europe** to uncover insights, improve decision-making, and identify business opportunities.

The workflow covers the full data analysis lifecycle — from raw data cleaning to building an interactive dashboard.

## 🎯 Objectives
- Analyze pharmacy performance across multiple European locations  
- Identify trends in sales, products, and customer behavior  
- Improve data quality through cleaning and transformation  
- Build a dynamic dashboard for business insights  

---

## 🛠️ Tools & Technologies
- **SQL** → Data extraction and restructuring  
- **Python** → Data cleaning, preprocessing, and analysis  
- **Power BI** → Dashboard creation and data visualization  
- **DAX** → Advanced calculations and measures  

---

## 🔄 Data Processing Workflow

### 1. Data Extraction (SQL)
- Queried and rebuilt tables from raw datasets  
- Structured data for better usability  

### 2. Data Cleaning & Transformation
- Removed duplicates and handled missing values  
- Standardized formats across all branches  
- Created new calculated columns to improve analysis  

### 3. Data Analysis (Python)
- Explored patterns and trends  
- Performed data validation and deeper analysis  

### 4. Data Modeling & DAX (Power BI)
- Built relationships between tables  
- Created measures using DAX for KPIs and metrics  

### 5. Dashboard Development
- Designed interactive visuals  
- Enabled filtering by country, branch, and product
